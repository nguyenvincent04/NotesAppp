## Project Description

**NotesApp** is an Android application that allows users to create, edit, delete, and manage notes efficiently.  
The application is developed using **modern Android development practices** and demonstrates concepts covered from **Week 1 through Week 15** of CPSC 411A.

This app serves as the **Final Project** for the course.

---

## Project Objectives

- Apply Android fundamentals learned throughout the semester  
- Use Kotlin and Jetpack Compose for UI development  
- Implement state management and navigation  
- Store user data using local persistence  
- Follow Material Design guidelines  
- Demonstrate MVVM architecture  

---

## Features

- Account Login/Logout (Firebase)
- Create new notes  
- Edit existing notes  
- Delete notes  
- View notes in a scrollable list  
- Material Design 3 UI  
- Persistent local storage  
- Multi-screen navigation
